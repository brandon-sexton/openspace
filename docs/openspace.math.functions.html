TOLERANCE: float = 1e-12¶

the tolerance used to stop the recursive solutions of Kepler’s equation

static from_ma_e(ma: float, e: float) → float¶

calculate the eccentric anomaly using algorithm 2 in Fundamentals of Astrodynamics and Applications, 4th Edition

\[E_{n+1} = E_n + \left(\frac{M-E_n+e\sin{(E_n)}}{1-e\cos{(E_n)}}\right)\]

Note

\(E_0 = M - e\) for \(-\pi<M<0\) or \(M>\pi\) otherwise \(E_0 = M + e\)

looping continues until \(\vert E_{n+1} - E_n\vert < tolerance\)

Parameters:

  • ma (float) – mean anomaly in \(rads\)

  • e (float) – eccentricity

Returns:

eccentric anomaly in \(rads\)

Return type:

float
