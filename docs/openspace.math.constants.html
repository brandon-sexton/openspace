openspace.math.constants¶

BASE_IN_KILO: float = 1000¶

number of base units in a kilo in \(\frac{(base)}{kilo(base)}\)

DEGREES_IN_HOUR: float = 15¶

number of degrees equal to one hour in \(\frac{\deg}{hr}\)

HOURS_IN_DAY: float = 24¶

number of hours in one solar day in \(\frac{hr}{day}\)

MINUTES_IN_DAY: float = 1440¶

number of minutes equal to one solar day in \(\frac{min}{day}\)

MINUTES_IN_HOUR: float = 60¶

number of minutes equal to one hour in \(\frac{min}{hr}\)

SEA_LEVEL_G: float = 0.00981¶

gravity at sea level in \(\frac{km}{sec^2}\)

SECONDS_IN_DAY: float = 86400¶

number of seconds in one solar day in \(\frac{sec}{day}\)

SECONDS_IN_HOUR: float = 3600¶

number of seconds equal to one hour in \(\frac{sec}{hr}\)

SECONDS_IN_MINUTE: float = 60¶

number of seconds equal to one minute in \(\frac{sec}{min}\)

SECONDS_IN_SIDEREAL_DAY: float = 86164.090518¶

number of seconds for Earth to complete one rotation in \(sec\) (found on page 31 of Vallado 4th Edition)
